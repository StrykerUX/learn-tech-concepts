# User Experience (UX)

**User Experience** o **Experiencia de Usuario** es el conjunto de percepciones y respuestas que tiene una persona al usar un producto, sistema o servicio.

## ¿Qué es User Experience (UX)?

Imagina que entras a un **restaurante por primera vez** - desde cómo encuentras la puerta, qué sientes al entrar, si el menú es fácil de entender, si el mesero es amable, hasta si sales satisfecho. **UX es exactamente eso, pero para productos digitales**.

User Experience es todo lo que una persona experimenta cuando interactúa con un producto digital: una app, un sitio web, un software. No es solo cómo se ve, sino cómo se siente usarlo y si realmente le sirve al usuario para lograr lo que necesita.

**UX se enfoca en:**
- **¿Es útil?** - ¿Realmente resuelve un problema del usuario?
- **¿Es fácil de usar?** - ¿Puede la gente usarlo sin frustrarse?
- **¿Es accesible?** - ¿Pueden usarlo personas con diferentes capacidades?
- **¿Genera confianza?** - ¿Los usuarios se sienten seguros usándolo?
- **¿Es agradable?** - ¿La experiencia es placentera o al menos no molesta?

Un buen UX es invisible: cuando todo funciona tan bien que el usuario puede concentrarse en su objetivo sin pensar en cómo usar la herramienta.

## Elementos del UX

### 1. Arquitectura de Información
- **Organización** del contenido
- **Navegación** intuitiva
- **Etiquetado** claro
- **Sistemas de búsqueda**

### 2. Diseño de Interacción
- **Flujos de usuario**
- **Microinteracciones**
- **Retroalimentación** del sistema
- **Estados** de la interfaz

### 3. Diseño Visual
- **Jerarquía visual**
- **Tipografía**
- **Color** y contraste
- **Espaciado** y layout

## Proceso de Diseño UX

### 1. Investigación
```
Investigación de usuarios:
• Entrevistas
• Encuestas
• Análisis de competencia
• Personas (arquetipos)
• Journey maps
```

### 2. Definición
```
Definición del problema:
• Problem statements
• How Might We questions
• Objetivos del usuario
• Objetivos del negocio
```

### 3. Ideación
```
Generación de ideas:
• Brainstorming
• Sketching
• Wireframes
• User flows
```

### 4. Prototipado
```
Creación de prototipos:
• Prototipos de papel
• Prototipos digitales
• Prototipos interactivos
• High-fidelity mockups
```

### 5. Testing
```
Pruebas con usuarios:
• Usability testing
• A/B testing
• Heuristic evaluation
• Analytics y métricas
```

## Principios de UX

### 1. Utilidad
¿El producto resuelve un problema real?

### 2. Usabilidad
¿Es fácil y eficiente de usar?

### 3. Deseo
¿Genera emociones positivas?

### 4. Accesibilidad
¿Pueden usarlo todos los usuarios?

### 5. Credibilidad
¿Los usuarios confían en el producto?

### 6. Encontrabilidad
¿Los usuarios pueden encontrar lo que necesitan?

### 7. Valor
¿El producto aporta valor al usuario y al negocio?

## Métodos de Investigación

### Cualitativos
- **Entrevistas**: Conversaciones profundas
- **Observación**: Ver cómo usan el producto
- **Card Sorting**: Organización de contenido
- **Focus Groups**: Discusiones grupales

### Cuantitativos
- **Encuestas**: Datos de muchos usuarios
- **Analytics**: Comportamiento real
- **A/B Testing**: Comparación de variantes
- **Heatmaps**: Mapas de calor de interacción

## Herramientas UX

### Investigación
- **Google Analytics**: Métricas web
- **Hotjar**: Heatmaps y grabaciones
- **UserTesting**: Pruebas remotas
- **Typeform**: Encuestas

### Diseño
- **Figma**: Diseño colaborativo
- **Sketch**: Diseño de interfaces
- **Adobe XD**: Prototipado
- **Miro**: Mapas mentales

### Prototipado
- **InVision**: Prototipos interactivos
- **Principle**: Animaciones
- **Framer**: Prototipos avanzados
- **Marvel**: Prototipado simple

## Métricas UX

### Métricas de Usabilidad
- **Task Success Rate**: Éxito en tareas
- **Time on Task**: Tiempo para completar
- **Error Rate**: Tasa de errores
- **System Usability Scale (SUS)**: Escala de usabilidad

### Métricas de Engagement
- **Session Duration**: Duración de sesión
- **Pages per Session**: Páginas por sesión
- **Bounce Rate**: Tasa de rebote
- **Return Rate**: Tasa de retorno

### Métricas de Conversión
- **Conversion Rate**: Tasa de conversión
- **Funnel Analysis**: Análisis de embudo
- **Click-through Rate**: Tasa de clics
- **Cart Abandonment**: Abandono de carrito
